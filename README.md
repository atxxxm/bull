# Installation
## Linux
1. ```git clone https://github.com/atxxxm/bull```
2. ```chmod +x bull/install.sh```
3. ```./bull/install.sh```

## Windows
1. ```git clone https://github.com/atxxxm/bull```
2. ```cd bull```
3. ```install.bat```

# Requirements
- C++ 17
